# File Size

[![Hex.pm](https://img.shields.io/hexpm/v/file_size.svg)](https://hex.pm/packages/file_size)

A file size calculator, parser and formatter.

## Prerequisites

* Erlang 20 or greater
* Elixir 1.8 or greater
